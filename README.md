# 📈 Solar Pro 2 주식 뉴스 수집 및 요약기

네이버 증권의 종목 뉴스를 실시간으로 수집하고, **Upstage Solar Pro 2 API**를 사용하여 핵심 내용을 요약하는 경량 크롤러입니다. 중복 수집 방지(ID 기반) 및 일별 증분 저장 기능이 포함되어 있습니다.

## 🚀 주요 기능
- **수동 종목 입력**: 원하는 종목 코드(6자리)를 입력하여 즉시 수집 시작.
- **ID 기반 증분 수집**: 네이버 뉴스 고유 ID(oid, aid)를 대조하여 이미 수집된 기사는 건너뛰고 새 기사만 추가.
- **Solar Pro 2 요약**: 긴 기사 본문을 [논조], [관련종목], [핵심내용]으로 깔끔하게 한 문장 요약.
- **일별 데이터 관리**: `news/종목코드_YYYYMMDD.json` 형태로 매일 새로운 파일 생성 및 업데이트.

## 💻 윈도우 PC 실행 가이드 (Setup & Run)

다른 윈도우 PC에서 프로젝트를 가져와 실행할 때 아래 순서를 따르세요.

### 1. 환경 준비
*   **Python 설치**: [Python.org](https://www.python.org/)에서 3.10 버전 이상을 설치하세요.
*   **uv 설치**: PowerShell(파워쉘)을 열고 아래 명령어를 입력하여 파이썬 패키지 매니저 `uv`를 설치합니다.
    ```powershell
    powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
    ```
    *(설치 후 터미널을 껐다 다시 켜주세요.)*

### 2. 프로젝트 가져오기 및 설정
```bash
# 리포지토리 복제
git clone https://github.com/사용자아이디/리포지토리이름.git
cd 리포지토리이름

# 환경 변수 설정
copy .env.example .env

```

## ⏰ 크롤러 자동화 설정 (Windows Task Scheduler)

윈도우에서 일정 시간마다 수집기를 자동으로 가동하여 데이터를 쌓고 싶다면 아래 설정을 따르세요.

### 1. 작업 생성
1. **시작 메뉴**에서 `작업 스케줄러`를 검색하여 실행합니다.
2. 우측 메뉴에서 **기본 작업 만들기...**를 클릭합니다.
   - **이름**: `StockNewsCrawler`
   - **트리거**: `매일` 또는 `매주` 중 원하는 주기를 선택합니다. (작업 완료 후 정밀 설정이 가능하므로 우선 아무거나 선택합니다.)

### 2. 동작 설정 (uv 실행)
1. **동작**: `프로그램 시작`을 선택합니다.
2. **프로그램/스크립트**: `uv`의 절대 경로를 입력합니다.
   - 보통 위치: `C:\Users\사용자이름\.cargo\bin\uv.exe`
   - 경로를 모를 경우 터미널(CMD/PowerShell)에서 `where uv`를 입력해 확인하세요.
3. **인수 추가(옵션)**: `run main.py`를 입력합니다.
4. **시작 위치(옵션)**: **프로젝트 폴더의 전체 경로**를 입력합니다.
   - **중요**: 이 경로가 정확해야 프로그램이 `.env` 파일과 `news/` 폴더를 정상적으로 인식합니다.
   - 예: `C:\Users\Admin\Desktop\stock-summarizer`

### 3. 반복 주기 정밀 설정 (예: 30분마다 가동)
1. 작업 생성이 완료되면 목록에서 만든 작업을 **우클릭 -> 속성**으로 들어갑니다.
2. **트리거** 탭 클릭 -> 설정된 트리거 선택 후 **편집** 클릭.
3. **고급 설정**에서 **작업 반복 간격**을 `30분` 또는 `1시간`으로 변경하고, 기간을 **무기한**으로 설정합니다.

### 4. 완료
이제 윈도우가 켜져 있는 동안 정해진 시간마다 자동으로 뉴스를 긁어와 **Solar Pro 2**로 요약한 뒤 `news/` 폴더에 JSON 파일로 저장합니다.
